Présentation

Une fonction JavaScript permettant de rendre les title accessible au clavier

Comment ça fonctionne ?

Au chargement de la page le script génère une infobulle accesssible au clavier via la touche de tabulation pour tous les éléments, paramétrés dans le script, possédant un title

Démonstration

MFP VidéoPlayer

Le RGAA

Comment l'utiliser

Copier le script et l'appeller dans le head de la page : 


				<script type="text/javascript" src="AccessTooltip.js"></script>
